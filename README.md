# Criando-um-Dashboard-de-Vendas-do-Xbox-no-Excel-com-DIO

Você já pensou em criar um dashboard do zero e aprender a montar passo a passo?

Neste repositório, você vai construir um dashboard completo a partir de uma planilha de vendas do Xbox no Excel. Tudo isso com um guia textual simples, que explica cada etapa de forma clara para você acompanhar sem dificuldades.

O repositório já possui as planilhas necessárias:

uma planilha inicial para você começar

e uma planilha final para comparar o resultado

Assim, você aprende praticando e consegue evoluir no processo de criação de dashboards.

Bora tentar? 🚀
